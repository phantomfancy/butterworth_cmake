# butterworth_cmake

An embedded butterworth filter with zero-floating-point implementation.

## Introduction

一个支持二阶和三阶的巴特沃斯滤波器，零浮点运算实现，适用于低/不具备浮点计算能力的嵌入式设备。

A Butterworth filter that supports second-order and third-order, zero-floating-point implementation, suitable for embedded devices with low / no floating-point computing power.
